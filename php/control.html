Control Structures in PHP

Conditional Statements

Conditional statements in PHP allow you to make decisions in your code based on certain conditions. PHP supports several conditional statements:

If else Statement :Executes code if a specified condition is true otherwise else will be executed. Only if can be used. Example:

$age = 15;
if ($age < 18) {
echo "You are under 18.";
} else if ($age < 30) {
echo "You are under 30.";
} else {
echo "You are 18 or older.";
}

You are under 18

Switch StatementEvaluates a value against multiple possible cases. Example:

$day = "Monday";
switch ($day) {
case "Monday":
echo "It's Monday!";
break;
case "Tuesday":
echo "It's Tuesday!";
break;
}

It's Monday!

Looping Statements

Looping statements in PHP allow you to execute a block of code repeatedly. PHP supports several looping statements:

For Loop :Executes code a specified number of times. Example:

for ($i = 0; $i < 5; $i++) {
echo "Iteration: $i <br>";
}

Output:-

Iteration : 0
Iteration : 1
Iteration : 2
Iteration : 3
Iteration : 4

While loop :Executes code while a specified condition is true. Example:

$counter = 0;
while ($counter < 3) {
echo "Iteration: $counter <br>";
$counter++;
}

Iteration: 0
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

Do-While Loop :Executes code at least once and continues as long as a specified condition is true. Example:

$counter = 0;
do {
echo "Iteration: $counter <br>";
$counter++;
} while ($counter < 6);

Iteration: 0
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

Foreach Loop :Iterates over elements in an array or other iterable objects. Example:

$colors = array("red", "green", "blue");
foreach ($colors as $color) {
echo "Color: $color <br>";
}

Color: red
Color: green
Color: blue

Conclusion

Control structures are essential elements of PHP programming that allow you to make decisions, repeat tasks, and include external code. By mastering conditional statements and looping statements, you gain the ability to create dynamic and interactive web applications.
